# RoomKey lablab video script

Target length: 75–90 seconds.

00:00–00:10 — Problem: agent rooms are useful, but regulated teams cannot let sensitive context become permanent room history.
00:10–00:20 — Solution: RoomKey adds a disclosure gate to Band rooms. Agents can coordinate; protected context waits for a narrow human grant.
00:20–00:35 — Demo flow: Agent asks → Gate blocks → Human grants scope → Safe collaboration → Revoke + seal.
00:35–00:55 — Show the evidence console: protected values posted = 0, replay blocked, browser PASS, receipt hashes match.
00:55–01:10 — Show downloadable proof: proof pack, receipt JSON, attack matrix, evidence manifest.
01:10–01:25 — Boundary and value: receipt-pinned prototype, not production security; valuable for finance, healthcare, legal, compliance, and incident rooms.

One-sentence voiceover:
RoomKey lets Band agents collaborate on regulated work while sensitive context moves only after a scoped human grant, then leaves a browser-checkable receipt instead of a trust-me story.
